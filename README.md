# Aplicação CRUD básico utilizando Java e Spring Boot

Projeto básico de implementação do CRUD (Create, Read, Update, Delete) usando **Java** e **Spring Boot**. 

---

## 🚀 Ferramentas

- Operações CRUD
- Integração com o banco de dados **MongoDB**
---

## 🛠️ Tecnologias

- **Java 17**
- **Spring Boot**
- **MongoDB**
- **Maven**
---

## ⚙️ Requisitos
- **Java 17** ou superior
- **Maven**
